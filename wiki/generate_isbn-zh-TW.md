# generate_isbn

生成 ISBN 條碼，附帶校驗碼驗證和 ISBN-10/13 格式互轉。

## 參數

| 參數 | 類型 | 必填 | 預設值 | 說明 |
|------|------|------|--------|------|
| `isbn` | string | 是 | — | ISBN-10 或 ISBN-13（可含破折號） |
| `format` | `"png"` \| `"svg"` | 否 | `"png"` | 輸出格式 |
| `scale` | number (1-10) | 否 | `3` | 縮放倍率 |
| `includeText` | boolean | 否 | `true` | 是否顯示 ISBN 文字 |

## 功能特色

### 自動校驗

工具會自動驗證 ISBN 校驗碼（check digit）：
- ISBN-10: 加權和模 11 驗證
- ISBN-13: 交替加權和模 10 驗證

如果校驗碼不正確，回傳中會標示警告但仍會生成條碼。

### 格式互轉

- 輸入 ISBN-10 → 同時顯示對應的 ISBN-13
- 輸入 ISBN-13（978 開頭）→ 同時顯示對應的 ISBN-10
- 輸入 ISBN-13（979 開頭）→ 無法轉換為 ISBN-10（顯示 N/A）

### 破折號處理

輸入的 ISBN 會自動去除破折號和空格：
- `978-0-306-40615-7` → `9780306406157`
- `0-306-40615-2` → `0306406152`

## 範例

### ISBN-13

```json
{
  "isbn": "9780306406157",
  "format": "png"
}
```

### ISBN-10

```json
{
  "isbn": "0306406152",
  "format": "svg"
}
```

### 含破折號的 ISBN

```json
{
  "isbn": "978-986-123-456-7"
}
```

### ISBN-10 含 X 校驗碼

```json
{
  "isbn": "155404295X"
}
```

## 回傳格式

回傳包含兩個 content 項目：

1. **文字 metadata**：
```
ISBN Input: 9780306406157
Valid: Yes
ISBN-13: 9780306406157
ISBN-10: 0306406152
```

2. **條碼圖片**（PNG 或 SVG）

## 錯誤情況

- ISBN 長度不是 10 或 13 → 回傳 `isError: true` 並說明
- 校驗碼不正確 → 仍生成條碼，但 metadata 中顯示警告
